# Course 2: Manage Security Risks

## Overview

This course is part of the Google Cybersecurity Professional Certificate. It builds on foundational concepts and focuses on identifying, analyzing, and managing security risks in organizations.

## Course Goals

By the end of this course, you will be able to:

- Identify common threats, risks, and vulnerabilities to business operations  
- Understand how security frameworks and controls protect organizations  
- Explain the CIA Triad (Confidentiality, Integrity, Availability)  
- Use security tools like SIEM for monitoring and analysis  
- Apply playbooks to respond to cybersecurity incidents  
- Conduct basic security audits  

## Key Topics Covered

- Security frameworks (NIST, OWASP)  
- Risk management processes  
- Threats, risks, and vulnerabilities  
- Security controls and compliance  
- SIEM tools and event monitoring  
- Incident response playbooks  

## Skills Developed

- Risk mitigation  
- Event monitoring  
- Incident response  
- Security auditing  
- Security management  

## Tools and Technologies

- Security Information and Event Management (SIEM) tools  
- Open Web Application Security Project (OWASP)  

## Course Structure

This course includes the following modules:

1. Security domains  
2. Security frameworks and controls  
3. Introduction to cybersecurity tools  
4. Incident response and playbooks  

## Real-World Application

This course focuses on practical cybersecurity skills such as:

- Monitoring systems for threats  
- Analyzing logs and security events  
- Responding to incidents using structured procedures  
- Conducting security audits  

## Key Takeaway

Managing cybersecurity risk is essential for protecting systems, data, and business operations in modern organizations.
