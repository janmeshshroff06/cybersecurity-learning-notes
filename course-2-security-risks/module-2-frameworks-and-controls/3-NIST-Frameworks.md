## Module 2.3: NIST Frameworks

## Overview

NIST develops cybersecurity frameworks to help organizations manage risk. These frameworks are used by businesses, government agencies, and nonprofits, and provide standards, guidelines, and best practices.

## Purpose of NIST Frameworks

NIST frameworks help organizations:

- Identify risks  
- Protect systems  
- Detect threats  
- Respond to incidents  
- Recover from attacks  

## NIST Cybersecurity Framework (CSF)

- A voluntary framework used worldwide  
- Helps improve an organization’s security posture  
- Provides a structured way to manage cybersecurity risk  

## 5 Core Functions of the NIST CSF

### 1. Identify

- Understand systems, assets, and risks  

Purpose:

- Know what needs protection  

Example:

- Identifying a compromised workstation  

### 2. Protect

- Apply safeguards to reduce risk  

Includes:

- Policies  
- Security tools  
- Employee training  

Example:

- Blocking an unauthorized device  

### 3. Detect

- Monitor systems to find threats quickly  

Purpose:

- Identify suspicious activity early  

Example:

- Security tools flagging unusual behavior  

### 4. Respond

- Take action during a security incident  

Includes:

- Containing threats  
- Investigating causes  
- Documenting incidents  

Example:

- Determining how an attack happened  

### 5. Recover

- Restore systems after an incident  

Purpose:

- Return to normal operations  

Example:

- Restoring lost data or repairing systems  

## Real-World Scenario

Example: unknown USB compromises a system  

- Identify: find affected system  
- Protect: block the device  
- Detect: check for additional threats  
- Respond: investigate the cause  
- Recover: restore files and systems  

## NIST SP 800-53

- A detailed catalog of security controls  
- Primarily used for:
  - Government systems  
  - Organizations handling federal data  

Focus areas:

- Confidentiality  
- Integrity  
- Availability  

## Why NIST Frameworks Matter

- Provide a clear structure for managing cybersecurity  
- Help organizations:
  - Respond quickly to incidents  
  - Reduce damage and downtime  
  - Improve long-term security  

## Key Takeaways

- NIST frameworks guide organizations through a full security lifecycle  
- The CSF provides a structured and repeatable approach to managing risk  
- Continuous monitoring and improvement are essential  

## Big Picture

Cybersecurity is about managing risk continuously. Frameworks like NIST help organizations improve security over time through structured processes.
