## Linux Navigation Commands

### Core Navigation Commands

* Important commands included:

  * `pwd`
  * `ls`
  * `cd`

## Purpose

* Used to:

  * Navigate directories
  * Locate files
  * Explore the Linux filesystem

## Reading and Managing Files

### File Reading Commands

* Important commands included:

  * `cat`
  * `head`

## Purpose

* Used to:

  * Read file contents
  * Preview logs
  * Analyze reports

## Filtering Content in Linux

### grep Command

* `grep` searches files for:

  * Specific strings or patterns

## Piping

* The pipe operator (`|`) sends output from one command into another command
* Helps combine commands for advanced filtering

## Cybersecurity Uses

* Log analysis
* Searching reports
* Investigating suspicious activity

## Managing Files and Directories

### Important Commands

* `mkdir` → Create directories
* `rmdir` → Remove empty directories
* `touch` → Create files
* `rm` → Delete files
* `mv` → Move or rename files
* `cp` → Copy files

## nano Text Editor

* Used to create and edit files directly in the shell

## File Permissions and Ownership

### Permissions

* Linux controls access using:

  * Read (`r`)
  * Write (`w`)
  * Execute (`x`)

## Permission Categories

* User
* Group
* Other

## Important Commands

* `chmod` → Change permissions
* `chown` → Change ownership

## Managing Users in Linux

### User Management Commands

* `useradd`
* `usermod`
* `userdel`

## Purpose

* Used to:

  * Add users
  * Modify accounts
  * Remove users
  * Manage group access

## sudo and Administrative Access

### sudo Command

* `sudo` allows users to run commands with elevated privileges

## Security Importance

* Should be used responsibly because it provides:

  * Administrative/system-level access

## Getting Help in Linux

### Linux Help Commands

* `man`
* `whatis`
* `apropos`

## Purpose

* Used to:

  * Learn commands
  * Understand options
  * Troubleshoot issues
