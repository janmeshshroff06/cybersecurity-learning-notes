## Man Pages Within the Shell

## Overview

- Linux provides built-in help resources directly within the shell

- These resources help users:
  - Learn commands
  - Understand options
  - Troubleshoot problems

## Important Help Commands

### man Command

- `man` stands for:
  - Manual

- Displays detailed information about:
  - Commands
  - Syntax
  - Options

## man Syntax

Example:

```bash id="f1klfy"
man [command]
```

## man Example

Example:

```bash id="b8tv7d"
man usermod
```

## Information Provided by man

- Command descriptions
- Syntax
- Flags/options
- Usage examples

## whatis Command

## Overview

- `whatis` provides:
  - A short one-line description of a command

## whatis Syntax

Example:

```bash id="6u1o8v"
whatis [command]
```

## whatis Example

Example:

```bash id="1st17t"
whatis tail
```

## Purpose

- Useful when users need:
  - A quick explanation
  - A fast reference

## apropos Command

## Overview

- `apropos` searches manual page descriptions for keywords

- Useful when users:
  - Do not know the exact command name

## apropos Syntax

Example:

```bash id="r53b7g"
apropos [keyword]
```

## apropos Example

Example:

```bash id="tr02ck"
apropos password
```

## Filtering apropos Results

Example:

```bash id="v4y9aw"
apropos -a change password
```

- `-a` returns results containing:
  - Both keywords
