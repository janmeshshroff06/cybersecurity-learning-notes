## Module 2.2: Linux Distributions

## Linux Distributions

## Overview

- Linux has many customized versions called:
  - Linux distributions
  - Distros

- Different distributions are designed for different:
  - Users
  - Tasks
  - Security needs

## Purpose of Linux Distributions

- Different distros focus on:
  - General computing
  - Security
  - Servers
  - Penetration testing
  - Digital forensics

- Users choose distributions based on:
  - Tools
  - Features
  - Performance needs

## Parent Distributions

- Many Linux distributions are based on:
  - Parent distributions

Examples:

- Debian → Ubuntu and KALI LINUX™
- Red Hat® → CentOS
- Slackware® → SUSE®

## Customization

- Linux distributions can include different:
  - Applications
  - Utilities
  - Interfaces
  - Security tools

- Linux is highly customizable because it is open source

## KALI LINUX™

## Overview

- KALI LINUX™ is a Linux distribution designed for cybersecurity professionals
- Commonly used for:
  - Penetration testing
  - Security analysis
  - Digital forensics

## Virtual Machines

- KALI LINUX™ is often installed on:
  - Virtual machines (VMs)

Benefits:

- Safe testing environment
- Easier recovery
- Prevents damage to the main OS

## Penetration Testing Tools

### Metasploit

- Exploits and tests vulnerabilities

### Burp Suite

- Tests web application security

### John the Ripper

- Password-cracking tool

## Digital Forensics Tools

### tcpdump

- Captures network traffic

### Wireshark

- Analyzes network packets

### Autopsy

- Investigates digital evidence

## Package Managers

## Overview

- Package managers help manage software in Linux
- Used to:
  - Install applications
  - Update software
  - Remove packages
  - Manage dependencies

## Benefits of Package Managers

- Simplifies software installation
- Automatically handles required dependencies
- Makes updates and maintenance easier

## Examples

### Debian-based Systems

- APT

### Red Hat®-based Systems

- YUM
- DNF

## Dependencies

- A dependency is software required for another program to function properly
- Package managers automatically install needed dependencies

## Key Takeaways

- Linux distributions are customized versions of Linux designed for different purposes
- KALI LINUX™ is widely used in cybersecurity

- Linux skills are essential in cybersecurity and system administration
