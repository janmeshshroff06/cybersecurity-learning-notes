## Module 1.4: Review – Introduction to Operating Systems

## Overview

* This section reviews the major concepts from Module 1
* Focuses on:

  * Operating systems
  * System operations
  * Resource management
  * Virtualization
  * User interfaces

## Operating Systems Recap

* An operating system (OS) connects:

  * Users
  * Applications
  * Hardware

* Common operating systems:

  * Windows
  * macOS
  * Linux

### Purpose of an OS

* Manage hardware and software resources
* Run applications efficiently
* Support multitasking
* Provide security features

## Boot Process Recap

### Startup Components

* BIOS or UEFI starts the system
* Bootloader loads the operating system

### Security Importance

* Vulnerabilities during boot process can expose systems to attacks

## System Operations Recap

### Applications and System Calls

* Applications send requests to the OS using system calls

### Hardware Communication

* OS communicates with:

  * CPU
  * Memory
  * Storage
  * Devices

## Resource Allocation Recap

### Resources Managed by OS

* CPU
* Memory
* Storage
* Bandwidth

### Multitasking

* OS balances resources between running applications

### Task Manager

* Displays:

  * Running processes
  * Resource usage

* Helps detect performance issues and suspicious activity

## Virtualization Recap

### Virtual Machines (VMs)

* Software-based computers running inside another system

### Hypervisors

* Manage virtual machines

### Cybersecurity Uses

* Malware analysis
* Safe testing environments
* Sandboxing suspicious files

## GUI vs CLI Recap

### GUI (Graphical User Interface)

* Uses icons and menus
* Easy for beginners

### CLI (Command-Line Interface)

* Text-based interface
* More powerful and efficient
* Important for cybersecurity professionals

## Cybersecurity Importance

* Security analysts use operating systems to:

  * Manage permissions
  * Monitor logs
  * Configure systems
  * Detect suspicious behavior

## Key Takeaways

* Operating systems manage communication between users, applications, and hardware
* Boot processes and resource management are critical system functions
* Virtualization improves efficiency and security testing
* GUI and CLI provide different ways to interact with systems
* Linux and CLI skills are especially valuable in cybersecurity

## Big Picture

* Operating systems form the foundation of modern computing and cybersecurity
* Control how systems function, communicate, allocate resources, and defend against threats
